# DbCloner

This project is a pair programming practice .
Изучаем базы данных - часть 2


Описание:


Требуется разработать консольную программу на Java которая будет переносить 
данные из одной базы в другую. Переносить - это создавать такую же структуру 
таблиц как в базе источник и заполнять данными из базы источника. Программе 
при запуске должен передаваться в виде аргумента путь к файлу с входными данными.


Формат входного файла:


Тип базы данных источника
Хост источника
Порт источника
Наименование базы источника
Пользователь источника
Пароль источника
Тип базы данных цели
Хост цели
Порт цели
Наименование базы цели
Пользователь цели
Пароль цели


Каждая строчка - один параметр. Тип базы данных может принимать два значения: 
MySql и PostgreSql, в любом регистре.


Условия:


1)Тип базы данных источника и цели различаются
2)Типы данных источника и цели должны соответствовать друг другу


Пример:


Пример входного файла


MySql
localhost
3306
testBd
root
1234
PostgreSql
localhost
5233
goal
postgres
qwerty


Дополнительные задания:


Помимо таблиц перенести хранимые процедуры и функции. Стоит учитывать что 
их синтаксис в разных базах различается.
